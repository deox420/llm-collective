# prototypes

Prototipos de exploracion. **No son el codigo final**, sirven de referencia.

- `CouncilScene.placeholder.jsx` — vista interactiva de Council con pixel-art dibujado
  en canvas (placeholders, sin assets de PixelLab). Demuestra el mapa etapa->pose,
  el clic->detalle y la barra por etapas. En Fase 6 se sustituyen los rectangulos por
  los sprites de PixelLab manteniendo el mismo contrato (ver `docs/13-interactive-scenes.md`).
