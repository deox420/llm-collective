# Second Brain

RAG sobre un vault de Obsidian. Ver SDD seccion 3.3.

- indexado: notas .md -> embeddings locales -> base vectorial
- consulta: recupera notas y el chairman responde citandolas
- acceso remoto via tunel seguro (Tailscale / Cloudflare)
- privacidad: chairman + vector DB en la misma GPU => las notas no salen
